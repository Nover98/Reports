skipLibCheck - заставляет компилятор пропускать проверку всех файлов .d.ts, где бы они ни находились: в node_modules, в корне проекта или в твоей папке types/.

Это нужно в основном чтобы:
- ускорить проверку проекта
- не тратить время на диагностику .d.ts
- скрыть ошибки в сторонних типах

Например, если в одном файле есть дубликаты

```js
export type Subscription = {
    id: string;
};

export type Subscription = {
    id: number;
}
```

то ошибки не будет, при импорте этого типа ts просто возьмёт какой-то из них.
