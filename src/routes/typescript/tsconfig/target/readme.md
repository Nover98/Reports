**target** - определяет в какую версию javascript превратится typescript.

es6/es2015
es2016
es2017
es2018
es2019
es2020
es2021
es2022

es2024

target связан с другими настройками, такими как module и moduleResolution. Если выбирать новейшие версии js в target и при этом не выставлять свежие опции в этих настройках, тс начинает выбрасывать ошибки.
