forceConsistentCasingInFileNames - есть файловые системы, которые не чувствительны к регистру в названиях файлов. Например можно импортировать файл с названием fileManager.ts указывая импорт из FileManager.ts. Так вот этот флаг заставляет компилятор показывать ошибку, если файл пытаются импортировать через имя, которое отличается от того, с которым он записан на диск. Другими словами заставляет TypeScript проверять, что регистр в import-пути совпадает с реальным именем файла.
