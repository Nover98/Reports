resolveJsonModule - разрешает typescript импортировать и типизировать json файлы. По умолчанию он этого делать не даёт. То есть если я импортирую json файл имея этот флаг, а потом пытаюсь вызвать у строки из json какой-нибудь метод для числа, то увижу ошибку типов от ts.

флаг resolveJsonModule может быть установлен в true, только если настройка moduleResolution не стоит в classic. Если попытаться скомпилировать код, выходит что ts либо компилирует в CommonJs, где жсон подхватывается через requre() и там, разумеется, это нормально работает. Либо тогда надо выбирать bundler, а там при компиляции тс сам ничего не добавляет, чтобы можно было читать жсон, рассчитывая, что этим займётся бандлер.
