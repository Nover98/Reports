**allowJs** - разрешает импортировать js файлы в ts файлы. 

При импорте без этой настройки покажет ошибку.

Could not find a declaration file for module './demo.js'. 



Если allowJs: false, то VS code, перестаёт связывать js файлы с контекстом проекта, а именно папки с файлом tsconfig.json внутри, это может быть как в самом корне проекта, так и где-то в подпапке. В этом случае VS code начинает применять к этим файлам настройки своего дефолтного конфига для TS Server и начнёт проверять типы и правильность кода. Не смотря на то, что checkJS может быть false.
