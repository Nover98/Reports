**esModuleInterop** — если стоит true, TypeScript улучшает совместимость между ESM-синтаксисом импорта и CommonJS-модулями. Благодаря этому старые библиотеки, которые экспортируются через module.exports, часто можно импортировать более удобным способом, например через import x from '...'. При компиляции TypeScript при необходимости добавляет вспомогательный interop-код (маленькие вспомогательные функции-хелперы, которые сами заворачивают содержимое модуля в объект с ключем default), чтобы такой импорт работал корректнее.
