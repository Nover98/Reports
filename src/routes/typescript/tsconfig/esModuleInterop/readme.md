**esModuleInterop** — это настройка, которая позволяет импортировать библиотеки с модульной системой не ESM, такие как CommonJS/AMD/UMD, будто это современные модули. То есть появляется возможность импортировать из что-то из таких библиотек используя стандартный синтаксис ESM, типа import React from 'react'. Это частично можно делать и без typescript, но если этот флаг true, то TypeScript при компиляции добавляет в код маленькие вспомогательные функции (хелперы), которые сами заворачивают содержимое модуля в объект с ключем default. Если библиотека экспортируется старым образом через module.exports = то можно будет импортировать вещи через esm удобнее.
